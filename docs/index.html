SportSee

Project 12 of the OpenClassrooms "Front End Developer" course. Sportsee is an application for monitoring your physical activity.

Objective

The goal of this project is: Develop an analytics dashboard with React.

Installation

Use the package manager git to download SportSee app.

$ git clone git@github.com:loicantoniak/LoicAntoniak_12_30042022.git

Usage

$ cd sportsee/
$ npm install
$ npm start

Runs the app in the development mode. Open http://localhost:3000 to view it in the browser.

The page will reload if you make edits. You will also see any lint errors in the console.

To obtain a production version do :

$ npm build

API

The repository containing all the source code to run the micro API for the SportSee sports analytics dashboard.
